# PEO Pricing Declaration
## Canonical Tier Amounts — All Phases

This document closes open item O-006 and establishes the binding pricing tiers for the Peak Equity Optimizer (PEO) product. No agent may publish or implement pricing-linked UI that contradicts these amounts without Founder approval and a version update to this document.

---

## Tier Table

| Tier | Price | Billing | What It Unlocks |
|------|-------|---------|-----------------|
| Free Estimator | $0 | — | Manual-input calculators (Seller net proceeds, Flip profit, Wholesale spread, BRRRR preview, Rehab budget band, Offer rule explainer). No live data, no comps, no verified ARV. |
| Seller (paid) | $49 / property | Per property (one-time per analysis) | Auto property facts, comp-backed value positioning, market context, live data notes, confidence score, seller readiness plan, path comparison with live comps, seller report PDF/export. |
| Investor Basic | $99 / month | Subscription | Auto-acquired property facts, VERIFIED ARV (median of qualified sold comps), MARKET ARV reference, sold comps list, rehab budget summary, MAO/profit/ROI/stress case, risk bands, recommendation, summary report export. |
| Investor Advanced | $299 / month | Subscription | All Basic features plus comp controls, line-item rehab budgeting, regional multipliers, multi-structure financing, DSCR/refi analysis, scenario planner (base/upside/downside), wholesale spreads, cash-on-cash, advanced appendices, full export controls. |
| Vendor (marketplace) | $199 / month | Subscription | Verified vendor profile, lead management dashboard, review responses, marketplace analytics, inbound seller/investor lead routing. |

## Commercial Control

- All public pricing pages, upgrade prompts, and paywalls must reference the exact amounts above.
- Promotional or introductory pricing requires Founder approval and must be documented in this file or a Founder-signed addendum.
- Regional pricing (Colombia, Jamaica, or other markets) requires explicit Founder approval before implementation.

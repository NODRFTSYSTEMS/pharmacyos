# 11. Template Usage Guide
## 11.1 When to Use Each Template
## 11.2 Template Storage Locations
Airtable: Client Intake Form (live form)
Notion: Project Brief, QA Checklist, Post-Mortem, Handoff Package
Google Drive /templates: SOW, Proposal, Change Order (Word templates)
GitHub /prompts: Prompt Engineering (Markdown files)
## 11.3 Template Maintenance
Quarterly review of all templates for relevance and accuracy
Update templates based on post-mortem learnings
Version all changes with clear changelog
Notify team of template updates via #templates Slack channel
Archive outdated templates with deprecation notice
